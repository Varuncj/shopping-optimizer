About the Algorithms

Knapsack: Finds the optimal solution with the highest total value while respecting weight and price constraints.

BFS: Explores all combinations level by level, finding all valid solutions.

DFS: Explores combinations by going deeper before backtracking, finding all valid solutions.

Hash Maps: Used for efficient item lookup and combination storage.
